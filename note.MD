--- CHEAT SHEET ---

localhost ou 127.0.0.1 :
une adresse IP locale à mon ordinateur à laquelle je peux branché des services sur les ports comme un server web sur le port 8080

HTTP (Hypertext Transfer Protocol) :
un protocole sous forme de requête–réponse (souvent sur TCP) où un client envoie une requête (URL, méthode comme GET/POST, en-têtes, corps) à un serveur web, qui répond avec des code de statuts (200, 404…), des en-têtes et un contenu

TCP (Transmission Control Protocol) :
un protocole de transport qui crée une connexion entre deux machines et garantit que les données arrivent fiablement, dans l’ordre et sans pertes

Socket (chaussettes ahahah) :
un descripteur identifié par IP + port + protocole auquel un programme se connecte pour envoyer/recevoir des données
